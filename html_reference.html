HTML Reference

The Basics

To start an HTML page you need to start with the "!DOCTYPE" tag, then the "html" tag. Next you need the "head" tag, then the opening and closing "title", "style" or any tag that belongs there depending on what you need to make your HTML page. After that, the "head" tag may be closed, then proceed to place the opening body tag and add whatever you want and/or need for your website. Once this is done, place the closing "body" and "html" tags.

Remember that almost every tag needs a closing tag to go with it. If you dont know how to make it a closing tag, all you have to do is put in a "/" as your second character in your second tag.

Tags

If you dont have tags you dont have a website. There are so many tags you can use for your website, such as the "title', "body", "head", "style", "strong", "div", and many more.

Tag Meaning Use
"!--...--" Defines a comment to comment in your work but not to be shown
"!DOCTYPE" Defines the document type to create different types of websites
"a" Defines a hyperlink to lead words to links
"body" Defines the document's body to make your website actually have content
"br" Defines a single line break to split lines
"col" Specifies column properties for each column within a "colgroup" element to make a column
"div" Defines a section in a document to make seperate everything from each other kind of
"footer" Defines a footer for a document or section to create a bottom text and end to the page
"h1"-"h6" Defines HTML headings to make headings for your paragraphs in different font sizes
"head" Defines information about the document to set the information about the website you are making
"html" Defines the root of an HTML document to define what type of website you are making
"img" Defines an image to add an image to your website
"li" Defines a list item to make a list
"ol" Defines an ordered list to make an ordered list
"p" Defines a paragraph to make paragraphs on your website
"source" Defines multiple media resources for media elements ("video" and "audio") to help find the image or video you are using
"style" Defines style information for a document to design your website
"table" Defines a table to create a table chart
"tr" Defines a row in a table to make rows in a list
"ul" Defines an unordered list to make an unordered list
